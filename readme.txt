Allow customers to personalize the products. It will allow customer to provide the printed Name, choose font, color and check a sample image.

== Description ==

This is an awesome plugin to quickly give your customers ability to personalize your products. People really like personalised items at their home especially children. This plugin add field like Printed Name, Font and Font Colors, Sample image etc.
You can easily install and setup these field from your Wordpress admin panel.
You can choose different labels for Printed Name.
You can add several font options in the backend which the customers will see when they will choose the personalisation checkbox on the product pages.
You can add several color options using the color picker in the backend.
